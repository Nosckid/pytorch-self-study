# 🔥 PyTorch Self-Study

> A structured, self-directed journey into deep learning and neural networks using PyTorch — built toward a long-term goal in healthcare AI and predictive diagnostics.

---

## 📌 About This Repository

This repository documents my hands-on learning journey through PyTorch, one of the most widely used deep learning frameworks in both research and production. Each notebook in this repository represents a chapter of that journey — progressing from foundational concepts to building and training neural networks from scratch.

The goal is not just to follow tutorials, but to build genuine, working competence in deep learning that can eventually be applied to real-world problems.

All notebooks are written and executed in **Google Colab**, organized by topic, and version-controlled here on GitHub as a living record of progress.

---

## 🎯 Learning Objectives

By the end of this self-study program, the aim is to be able to:

- Confidently work with tensors, gradients, and the PyTorch computation graph
- Build, train, and evaluate neural networks from scratch
- Understand and implement common deep learning architectures
- Apply deep learning to structured and unstructured data
- Work with real datasets
- Deploy trained models for inference in practical applications

## 🚀 How to Use This Repository

### Run in Google Colab (Recommended)
1. Navigate to any `.ipynb` file in the `PyTorch Self-Study/` folder
2. Click the **Open in Colab** button at the top of the file
3. Run cells sequentially from top to bottom
4. No installation needed — all dependencies are handled within each notebook

### Run Locally
If you prefer to run notebooks locally:

```bash
# Clone the repository
git clone https://github.com/your-username/pytorch-self-study.git
cd pytorch-self-study

# Install dependencies
pip install torch torchvision numpy pandas matplotlib scikit-learn jupyter

# Launch Jupyter
jupyter notebook
```

> **Recommended Python version:** 3.9 or higher

## 🧠 Why PyTorch?

PyTorch is the dominant framework in academic deep learning research and is rapidly expanding in industry, particularly in healthcare AI. Its dynamic computation graph, intuitive Pythonic syntax, and strong ecosystem — including MONAI for medical imaging — make it the right foundation for building AI systems in clinical and diagnostic contexts.

More practically: understanding PyTorch deeply means being able to build, debug, and modify models at a low level, rather than depending on black-box abstractions. That depth is what separates engineers who use AI tools from those who can build them.
